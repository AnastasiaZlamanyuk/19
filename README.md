## Описание
В ходе данной лабораторной работы было проведено знакомство с основами языка программирования JavaScript. Работа включала в себя сравнение ключевых концепций JS с языком C#, настройку рабочего окружения, а также выполнение практических заданий для закрепления материала.

### Что было изучено:
- Работа с циклами (`for`, `while`, `do...while`) и операторами `break`/`continue`.
- Создание и использование функций (классический способ, стрелочные функции, функции как значения).
- Основы работы с массивами (создание, добавление/удаление элементов, перебор).
- Основы работы с объектами (создание, свойства, методы, перебор).
